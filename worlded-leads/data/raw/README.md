# INEP raw ZIPs

Place the official Censo Escolar/Inep ZIP files here when you want GitHub Actions to process them on demand.

Expected names:

- `microdados_censo_escolar_2020.zip`
- `microdados_censo_escolar_2021.zip`
- `microdados_censo_escolar_2022.zip`
- `microdados_censo_escolar_2023.zip`
- `microdados_censo_escolar_2024.zip`
- `microdados_censo_escolar_2025.zip`

The parser also accepts equivalent ZIP names that contain the year, but the standardized names above make manual runs easier to audit.
